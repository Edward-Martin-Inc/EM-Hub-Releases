# EM Hub Releases

This repository is the **public distribution point** for [EM Hub](https://github.com/Edward-Martin-Inc/EM-Hub).

## Purpose

EM Hub's source code is maintained in a private repository. This repo exists solely to host public releases so that:

- End users can download the latest version of EM Hub
- The built-in auto-updater can check for and deliver new versions

## Structure

Each release contains the installable binaries for all supported platforms (Windows, macOS, Linux) along with a `latest.json` manifest consumed by the auto-updater.

## Do Not

- Push code, branches, or commits here — this repo holds releases only
- Manually edit or delete `latest.json` — it is auto-generated by the EM Hub release workflow
- Create releases manually unless the CI workflow is unavailable

## Release Process

Releases are published here automatically by the EM Hub CI workflow when a version tag (`v*`) is pushed to the source repository.
